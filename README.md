## Local Run
- Install dependencies
  ```
  npm install --force
  ```

- Run on [localhost:3000](http://localhost:3000)
  ```
  # mainnet
  npm run dev

  # staging
  npm run dev-dev

  # testnet
  npm run dev-testnet
  ```
